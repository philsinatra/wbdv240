Assignments:

Build First Pages

Due: Midnight the night before next class

Create a minimum of four html pages to be part of your final project (an online portfolio). Using ONLY HTML tags, populate as much content as possible on all of the pages.

Suggested pages:

  • index (home page)
  • about
  • portfolio
  • resume

Example: content on your 'about' page could include -Your current program of study at Drexel, why you chose your field of study, how far along you are in the program. What makes you unique? What do you like to do? What are your hobbies? A brief explanation of your past experience with web design (before this class), including HTML, XHTML, CSS, graphics, etc. (There is no need to impress me, just be honest.) If you have used any other web technologies or web development packages, include that also.

Make sure you are using your html tags properly and that they are properly nested. Do not worry about visual design, focus on adding content and HTML syntax/nesting.

All of your html pages will be linked together via a simple navigation list at the top of each page.

Site Map

Due: Midnight the night before next class

Create the first draft of your Site Map (textbook page 461). To build a Site Map, you need to know what is going to appear on your site, so spend time brainstorming. If you were going on a job interview and your instructor was your potential employeer, what would you want to show them in your portfolio? Consider building a brainstorm chart, showing what makes you different. Ask yourself the 5 Whys, http://en.wikipedia.org/wiki/5_Whys.

Once you have your brainstorming chart, build your Site Map. You can use paper and a pencil, or any software (Photoshop, Illustrator etc). If you draw on paper, scan your image so you can submit it online.

Site Map image should be:

  • 150dpi
  • 8.5 x 11
  • greyscale
  • .jpg file format

File name for your site map image should be in this format: wbdv240s###_jwt123_sitemap.jpg [replace ### with your class section number, replace jwt23 with your drexel email account prefix]

Build Wireframes

Due: Midnight the night before next class

Revisit your site map and make any additions/edits that are needed.

Build your first version of your Wireframes (text book page 463).

You should have a unique wireframe design for each of your web pages, even if the intended layout is similar or the same. Remember wireframes are simple sketches of how information will be laid out. What is the hierarchy?

You can use paper and a pencil, or any software (Photoshop, Illustrator etc). If you draw on paper, scan your image so you can submit it online.

Final images should be:

  • 150dpi
  • 8.5 x 11
  • greyscale
  • .jpg file format

Zip all files into one zip file. Make sure to rotate your images so they are easy to read, and enhance the levels/contrast if necessary.

File names for your wireframe images should be in this format: wbdv240s###_jwt123_wireframe_01.jpg [replace ### with your class section number, replace jwt23 with your drexel email account prefix. Update file number 01 as needed]
